How It Works

Pick a challenge. Write the best code. Get paid. Simple.

1

Browse challenges

New code bounty challenges are posted daily. Each one has a clear task, a bounty amount, and a deadline. Pick one that matches your skills.

2

Submit your code

Write your solution and submit it. Other people are submitting too — the best code wins. All submissions are timestamped and public.

3

Win the bounty

When the challenge closes, the best submission is selected as the winner. If that's you, you get paid the bounty amount immediately.

4

Get paid

No waiting around. The moment you win, the bounty is yours. Withdraw your earnings anytime. Keep winning to earn more.

Why Dojo

Get paid for writing good code. No subscriptions, no gatekeeping, no waiting.

Best code wins

Multiple people submit on the same challenge. The best solution takes the bounty. Merit-based, not first-come-first-served.

Flexible deadlines

Each challenge has its own timer — some close in hours, some in days, some when enough submissions come in. Pick what works for you.

Fair and transparent

Every submission is timestamped and publicly visible. No backroom deals — the best code is the best code, and everyone can see it.

Instant payout

Win a challenge, get paid immediately. No invoice, no waiting period, no minimum threshold. The bounty hits your account the moment you win.

Use any tools

Use any coding agent or write code by hand. Dojo doesn't care how you solve the problem — just that your code is the best.

Free to join

No subscription, no fees, no cut from your winnings. You keep 100% of every bounty you win. Sign up in 30 seconds.
